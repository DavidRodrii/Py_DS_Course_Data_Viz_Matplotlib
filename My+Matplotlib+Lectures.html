Object Oriented API Method¶

Instantiate figure objects then call methods or attributes from object.

So we create figure objects, then call methods to them

In [14]:
fig = plt.figure()
# think of figure object as blank canvas
# We can add a set of Axes by passing it as a List
# List takes in 4 Arguments: Left, Bottom, Width, Height
# Ranging from 0 - 1
axes = fig.add_axes([0.1,0.1,0.8,0.8])
# Now that we have the Axes made, we can plot on them
axes.plot(x,y)
axes.set_xlabel('X Label')
axes.set_ylabel('Y Label')
axes.set_title('Title')
Out[14]:
<matplotlib.text.Text at 0x10f71a5c0>

Object Oriented Method gives us much more control over our parameters, but it does take a few more steps

Legends¶

Adding a Legend is a two step process

step 1.) ax.legend( )

step 2.) define labels inside ax.plot( ) commands

ax.plot(x, x**2, label='X Squared')

linewidth = # is obvious, we can also type in lw = # for shorthand

If we want transparent lines, we can use the alpha = # argument to pass in a number between 0 and 1

linestyle = '--' is another argument that in this case indicates dashed lines, with ls = '--' for shorthand as well
